# Security Policy

## Supported Versions

The following versions of this boilerplate receive security updates:

| Version | Supported |
| ------- | --------- |
| 1.x.x   | ✅        |
| 0.x.x   | ❌        |

## Reporting a Vulnerability

If you find a security vulnerability, please report it privately at **security@yourdomain.com**.  
You’ll receive a confirmation within **48 hours**, and a detailed follow-up within **5 working days**.  
Accepted issues will be patched promptly, and you’ll be informed of the resolution status.
